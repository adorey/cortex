# 🚀 Cortex v0.1.0 — First Release

> *"Don't Panic."* — The first stable foundation of the Cortex AI team framework.

## What is Cortex?

Cortex is a **modular framework for orchestrating AI coding agents** with structured roles, personalities, and technical capabilities. It turns a generic AI assistant into a specialized, multi-expert team that follows your project's conventions.

## Highlights

### 🏗️ 3-Layer Architecture
- **Personalities** — Themeable character identities (ships with a complete *Hitchhiker's Guide to the Galaxy* theme: 15 characters)
- **Roles** — Stack-agnostic technical skill cards organized by domain (engineering, product, data, communication, security & compliance)
- **Capabilities** — Loadable, stack-specific best-practice files (PHP, TypeScript, Symfony, MySQL, PostgreSQL, MongoDB, Docker, Kubernetes, OWASP)

### 👥 15 Agent Roles
Architect · Lead Backend · Lead Frontend · Platform Engineer · QA Automation · DBA · Performance Engineer · Consultant Platform · Product Owner · Business Analyst · Data Analyst · Tech Writer · Security Engineer · Compliance Officer · **Prompt Manager** (the orchestrator)

Each role card includes: SYSTEM PROMPT with behavioral rules, responsibilities, universal principles, anti-patterns, naming conventions, PR/review checklists, capability loading instructions, and cross-role interactions.

### 🎭 H2G2 Personality Theme
15 fully-developed character cards with personality prompts, behavioral rules, communication style guides, and a structured 5-step approach per character. Theme-agnostic role cards ensure new themes can be added without touching the core.

### 🔄 Prompt Manager & Dispatch Protocol
A central orchestrator that analyzes every request, looks up matching workflows, dispatches to the right expert, loads stack-specific capabilities, and proposes archiving — with a complete archiving protocol (format, naming convention, append-never-overwrite).

### ⚙️ Setup & Tooling
- `setup.sh` — Automated bootstrap: copies templates, resolves personalities, injects VS Code settings
- `--tool` option — Multi-AI tool support (VS Code Copilot, Cursor, Windsurf...)
- Workspace mode — Works as either a Git submodule (monorepo) or a standalone clone (multi-repo workspace) sitting alongside independent service repos
- `project-context.md` + `project-overview.md` — Split context templates for stack/conventions and business vision

### 🔀 Workflows Layer
Generic and project-specific workflow templates for multi-agent orchestration, with priority resolution (project workflows override generic ones).

### 🌍 Full English Translation
All files translated from French to English for international accessibility.
