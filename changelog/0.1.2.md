# 🚀 Cortex v0.1.2 — Minimal Dependencies Principle

> *"I'd far rather be happy than right any day."* — A lean update that enshrines dependency discipline across all engineering roles.

## Highlights

### 📏 New Universal Principle: Minimal Dependencies
- Added to **all 4 engineering role cards**: Architect, Lead Backend, Lead Frontend, Platform Engineer
- Rule: *"Every dependency is a liability. Can the framework or standard library do it? Reject convenience-only packages."*
- Applies across all technologies and stacks — not just backend
